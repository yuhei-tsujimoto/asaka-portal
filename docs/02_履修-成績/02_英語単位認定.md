# 英語単位認定

外部英語試験（TOEIC、TOEFL、IELTSなど）のスコアに応じて、英語科目の単位を認定する制度です。

## 対象となる試験

- TOEIC L&R
- TOEFL iBT
- IELTS
- 英検

## 申請方法

1. 試験結果の公式スコアを用意
2. 学務課窓口で申請書を受け取り
3. 必要書類を添付して提出

## 申請期限

各学期の履修登録期間内

## 注意事項

- スコアの有効期限は試験日から2年以内
- 既に単位を取得した科目は認定対象外
- 詳細な認定基準は学務課にお問い合わせください
