# 公開臨海実習 / 学外実習

学外で行われる実習プログラムへの参加申請についての情報です。

## 公開臨海実習

海洋生物や海洋環境について、臨海実験所などで実施される実習プログラムです。

### 対象学年
2年生以上（実習により異なります）

### 実施時期
夏季休暇期間中が中心

### 申請方法
1. 掲示板や学務課で実施要項を確認
2. 指導教員に相談
3. 申請書を学務課に提出
4. 参加費の納入

## 学外実習

病院、研究機関、企業などで行われる専門実習です。

### 申請の流れ
1. 実習先の決定（指導教員と相談）
2. 実習計画書の作成
3. 学務課に申請書を提出
4. 実習先との調整
5. 必要書類の準備（誓約書、保険加入など）

## 注意事項

- 実習には学研災・学研賠への加入が必須です
- 実習先への交通費・宿泊費は自己負担の場合があります
- 実習期間中の授業欠席については事前に届出が必要です
