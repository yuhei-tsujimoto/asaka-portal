# Asaka Campus Portal

東洋大学朝霞キャンパスの学生向けポータルサイトです。学生生活に必要な情報をまとめています。

## 📚 コンテンツ

### スケジュール
年間行事予定や授業日程などの情報

### 履修・成績
履修登録、英語単位認定、大学院科目先行履修など

### 学生生活
奨学金、サークル、施設利用などの情報

### 進路・キャリア
就職支援、資格取得に関する情報

### 各種申請
証明書発行、休学・退学、在留資格など各種手続き

### 施設・設備
トレーニングルーム、充電BOX、学食など施設の利用案内

## 🔍 サイトの使い方

左側のメニューから、知りたい情報のカテゴリを選択してください。各ページには詳細な手続き方法や利用案内が記載されています。

## 📞 お問い合わせ

ご不明な点は、朝霞キャンパス学務課までお問い合わせください。
